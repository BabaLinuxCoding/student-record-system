# Student Record System

A simple Bash script-based project to manage student records efficiently from the Linux terminal.

## 🚀 Features
* Add new student records.
* View and list all registered students.
* Search for a student by ID or Name.
* Automated data persistence using text files.

## 📁 File Structure
* `project.sh` - The main executable shell script containing the logic.
* `students.txt` - The database file where all student records are stored.

## 💻 How to Run
To run this project in your local Linux / Kali Linux machine, follow these steps:

1. Open your terminal and navigate to the project directory.
2. Give execution permission to the script:
   ```bash
   chmod +x project.sh
